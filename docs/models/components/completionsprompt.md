# CompletionsPrompt


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `string[]`

```typescript
const value: string[] = [
  "<value>",
];
```

### `number[]`

```typescript
const value: number[] = [
  986366,
];
```

### `number[][]`

```typescript
const value: number[][] = [
  [
    951736,
  ],
];
```
